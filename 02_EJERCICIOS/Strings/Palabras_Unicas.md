# 🧩 Reto 01: Palabras Únicas

### Enunciado
Dada una cadena de texto, devuelve el número de palabras que no se repiten.

### ⚠️ Restricciones / Notas
- No contar signos de puntuación.
- "Hola" y "hola" se consideran la misma palabra.
