# 🎨 Guía de Estilo y Formato para la Bóveda

Usa esta guía de referencia para mantener nuestras notas teóricas y retos de código con una estructura visual limpia, profesional y uniforme.

---

## 1. Enlaces y Multimedia (Sintaxis Básica)

| ==Qué quieres lograr==            | ==**Comando / Sintaxis**==          | ==**Resultado**==                 |
| :-------------------------------- | :---------------------------------- | :-------------------------------- |
| **Enlace normal**                 | `[[Nombre_Archivo]]`                | [[Nombre_Archivo]]                |
| **Texto personalizado (Alias)**   | `[[Nombre_Archivo\|Texto visible]]` | **Texto visible**                 |
| **Incrustar imagen o Excalidraw** | `![[pizarra.excalidraw]]`           | *(Se despliega el lienzo/imagen)* |
| **Texto en Negrita**              | `**Texto importante**`              | **Texto importante**              |
|                                   |                                     |                                   |
>[!Tip] Nota
Lo anterior es para **enlazar notas**, ahora si quieres crear un hipervínculo de una página usa lo siguiente:             `[Ir a Google](https://google.com)` y el resultado será el: [Ir a Google](https://google.com)


#### También podemos traer imagenes externas con una URL de la siguiente forma:
```
![Texto alternativo](https://moure.dev/logo.svg "Título opcional")
```

#### El resultado:
![Texto alternativo](https://moure.dev/logo.svg "Título opcional")
